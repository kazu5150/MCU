## 🎬 MCUとは

マーベル・シネマティック・ユニバース（MCU）は、マーベル・スタジオが製作する世界初の本格的なクロスオーバー映画シリーズです。

### インフィニティ・サーガ（フェーズ1-3）
2008年の『アイアンマン』から2019年の『アベンジャーズ/エンドゲーム』まで、23作品で構成される壮大な物語。6つのインフィニティストーンを巡る戦いが描かれます。

### マルチバース・サーガ（フェーズ4-6）
新世代のヒーローたちとマルチバースの概念を中心とした新たな物語が展開中。
